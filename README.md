# LocalChantMessenger
ローカルマシンのCLI上で簡易的なチャットを実行できます。

### 使用方法
* クライアントとサーバーのモジュールを分けているのでそれぞれで `go run main.go [username(任意)]` を実行。
* 起動後クライアントのCLI内で自由にメッセージを入力してください。
* 複数クライアントの同時接続も可能なため、ローカルマシン内で複数のシェルを立ち上げて実行可能です。
